# Youtube Comment Tool
Video comment filtering

### 1. Clone repo

```
git clone https://github.com/yemecen/yt-comment-tool.git
cd yt-comment-tool
```

### 2. Create .env

Create **.env** file. 

.env file content:
REACT_APP_API_TOKEN = Youtube Api key

### 3. Run 

```
$ npm install
$ npm start
```
